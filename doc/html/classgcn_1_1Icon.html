void gcn::Icon::draw ( Graphics *  graphics)
virtual

Draws the widget. It is called by the parent widget when it is time for the widget to draw itself. The graphics object is set up so that all drawing is relative to the widget, i.e coordinate (0,0) is the top-left corner of the widget. It is not possible to draw outside of a widgets dimension.

Parameters
graphicsa Graphics object to draw with.

Implements gcn::Widget.

void gcn::Icon::drawBorder ( Graphics *  graphics)
virtual

Draws the widget border. A border is drawn around a widget. The width and height of the border is therefore the widgets height+2*bordersize. Think of a painting that has a certain size, the border surrounds the painting.

Parameters
