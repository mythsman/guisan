Detailed Description

Used for grabbing user input and heavily used internally by Guichan. We include implemented Input classes for some common platforms like the Allegro library, the OpenGL library and the SDL library. To make Guichan usable under another platform, an Input class must be implemented.

See also
SDLInput, AllegroInput
